Project Overview

The Clinic Management System is a comprehensive web-based application developed using Python Django. It is designed to automate the manual processes of a medical clinic, managing doctors, patients, appointments, and billing efficiently.

This project is perfect for final-year Computer Science students looking for a robust Python Django project with source code. It includes an Admin Panel, Doctor Panel, and Patient Portal.

Key Features

  • Admin Dashboard: Manage doctors, patients, and appointments from a single view.
  • Online Appointment Booking: Patients can view doctor availability and book slots.
  • Doctor Management: Add, edit, or remove doctor profiles with specialization details.
  • Prescription Generation: Doctors can generate digital prescriptions.
  • Responsive Design: Works smoothly on mobile and desktop.

System Requirements

Language Python 3.8+
Framework Django 4.0
Database SQLite (Default) / MySQL
IDE VS Code / PyCharm

How to Run the Project?

Once you purchase the source code, follow these steps to run it on your machine:

  1. Download and extract the zip file.
  2. Open the folder in VS Code.
  3. Install requirements using: pip install -r requirements.txt
  4. Run migrations: python manage.py migrate
  5. Start server: python manage.py runserver
