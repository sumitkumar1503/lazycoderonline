Online Examination System in Java Spring Boot

Category: Java Enterprise Projects | Database: MySQL/H2

Java Spring Boot Thymeleaf MySQL

Project Overview

A complete Online Examination Portal where Admins can create tests with MCQs, and Students can take exams within a fixed time duration. The system automatically calculates scores and generates graphical reports.

Key Features

  • Exam Timer: Auto-submit exam when the timer ends.
  • Question Bank: Admin can add, edit, or delete MCQ questions easily.
  • Graphical Results: View student performance in bar charts and graphs.
  • Security: Spring Security login for Admin and Students.

System Requirements

  • Java JDK 17+
  • Maven
  • MySQL Database (or H2 in-memory)
